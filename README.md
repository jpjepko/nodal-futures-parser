# Parallelized Nodal Exchange EOD Futures Parser

## Requirements
`tabula-py` requires:
* Python >= 3.9
* Java >= 8

## Usage
Create virtual environment and install Python dependencies:
```sh
$ python3 -m venv env
$ source env/bin/activate
$ pip3 install -r requirements.txt
```

Run:
```sh
$ python3 main.py
```
